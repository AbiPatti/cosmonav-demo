### Select from Search Results
After hearing search results, you can select by:

- **"Choose option 1"** (or 2, 3, 4, 5)
- **"Select the first one"**
- **"Pick number 2"**
- **"Option 3"**

Works with: first, second, third, fourth, fifth, or numbers 1-5

---

### Repeat Search Results
- **"Repeat options"**
- **"List the results"**
- **"What are the options"**

Cosmo will read all search results again.

---

### Start Navigation
After selecting a destination:

- **"Start navigation"**
- **"Begin navigation"**
- **"Go"**

Cosmo will begin turn-by-turn navigation with hazard warnings.

---

### Stop Navigation
During active navigation:

- **"Stop navigation"**
- **"End navigation"**
- **"Cancel route"**

Cosmo will stop all navigation guidance.

## ğŸ’¡ Tips

- Always start commands with **"Hey Cosmo"** to activate the assistant
- Wait for Cosmo to finish speaking before giving the next command
- Speak clearly and at normal volume
- You can interrupt navigation at any time with voice commands
- The app works best with a good internet connection for transcription
- Hazard detection uses OpenStreetMap data (coverage may vary by location)
